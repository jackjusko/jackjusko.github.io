Hello Homelab: Starting the Journal

Jack Jusko
Software, AI & Cybersecurity Engineer
homelab meta security software-engineering

I spend a lot of time building and fixing things in my homelab. I already have pages for my bigger project writeups here, but I realized I missed having a place for the smaller stuff. Sometimes I just want to talk about a specific container config or a VLAN issue I finally sorted out.

Why I'm starting this

Engineering is mostly documentation. If I don't write it down, I end up searching for the same fix on StackOverflow six months later. I figure if I publish these notes, they might help someone else who hits the same edge cases.

I write about what I build and what I break in the lab. If this was useful, my portfolio has the longer-form project writeups, security labs, and work history.

See my portfolio →

What I'll write about

I plan to focus on software engineering, AI, and cybersecurity. For software, I'll share patterns and tools that actually work for me. On the AI side, I want to see what local models and agents can do without the hype. And for security, I'll post writeups for labs and how I'm hardening my own network.

I'm sticking with this dark terminal theme because it's what I'm used to. Welcome to the lab.

Thanks for reading. I am Jack Jusko — I build software, work in security, and document both on this site.
